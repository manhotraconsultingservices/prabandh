# Prabandh Screenshots — How to Add Real Application Screenshots

## Steps

1. Open https://prabandh.atithi-setu.com in Chrome
2. Log in with admin credentials
3. Take a full-page screenshot of each module (press F12 → device toolbar → set 1440px width)
4. Save each file in THIS folder with EXACTLY these names:

## Required Screenshot Files

| Filename | Module | URL to screenshot |
|---|---|---|
| `dashboard.png` | Dashboard — KPI Cards & Charts | /dashboard |
| `sales.png` | Sales Invoice List | /sales |
| `purchase.png` | Purchase Invoice List | /purchase |

| `timesheets.png` | Timesheets | /timesheets |
| `expenses.png` | Expenses | /expenses |
| `settings.png` | Settings / Company Config | /settings |

## Recommended Screenshot Tool
- **Chrome DevTools** → Ctrl+Shift+P → "Capture full size screenshot"
- Recommended size: 1440 × 900px minimum
- Format: PNG preferred (JPG also works)

## After Adding Screenshots
The gallery on the landing page will automatically show the images.
The lightbox click-to-expand will also work automatically.
No code changes needed.
